# Stick War Mini Game (Java OOP Project)

# Project Description
This is a simple game inspired by *Stick War Legacy*.  
It is built using Java and focuses on Object-Oriented Programming (OOP) concepts such as:

- Interfaces  
- Inheritance  
- Polymorphism  
- Encapsulation  
- ArrayLists  
- User input using Scanner  

The game allows users to create characters, choose classes, and simulate battles between them.

# Concepts i Used

- Interface (Combat): Defines shared actions like attack and displayInfo.

- Character class: Stores shared attributes like:
                  - name  
                  - health  
                  - damage  

- Inheritance : Swordsman, Archer, and Mage extend the Character class.

- Polymorphism : Each class has their own version of the attacking.

- Arraylist: Used to store multiple players.

- Scanner (User Input): Used to make the game interactive.

# Character Types

# Swordsman
- High health  
- Medium damage  
- sword attack  

# Archer
- Low health  
- Medium damage  
- Ranged attack  

# Mage
- Low health  
- High damage  
- Magic attack
